## Запуск проекта

Установить зависимости:

### `npm i`

Установить JSON-server:

### `npm install -g json-server`

Запустить сервер. Порт - 3004:

### `json-server --watch db.json --port 3004`

Запустить запустить проект:

### `npm start`
